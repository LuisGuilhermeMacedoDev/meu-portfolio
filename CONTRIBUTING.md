# Guia de Contribuição e Controle de Acesso

Este documento estabelece as regras para colaboração segura no PortfolioHUB, garantindo o controle de versão adequado.

## Fluxo de Trabalho (Workflow)
1. **Clonagem:** Faça o clone do repositório localmente.
2. **Branches:** O envio direto de código para a branch `main` está bloqueado por políticas de segurança. Crie uma nova ramificação para a sua funcionalidade.
3. **Commits:** Realize commits descritivos e organizados.
4. **Pull Requests (PR):** Envie sua ramificação para o servidor remoto e abra um Pull Request direcionado à `main`.
5. **Revisão:** O código só será integrado (merge) ao ambiente de produção após revisão e validação das regras de segurança estabelecidas.
